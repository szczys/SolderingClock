Soldering Clock

Clock built as part of [Clocks for Social Good](http://hackaday.com/2015/09/16/why-you-should-build-a-clock-for-social-good-this-week/).

Built from a repurposed SMD Soldering Practice board and an ATmega168

More details on my [Hackaday.io project page](https://hackaday.io/project/7697-soldering-clock)
